Для создания проекта использовал:
Django==1.8
pkg-resources==0.0.0
python 3.5.2

В поле "original" пишем текст который хотим зашифровать, затем в текстовом поле "step" указываем смещение.
 После чего нажимаем клавиши "shifr" и в поле "shifr_text" появляется зашифрованный текст.
 Также по нажатию клавиши "shifr" выводится граффик, который показывает частоту встречающихся символов в зашифрованном тексте.
 Граффик был взят с google-chart.
 Также текст можно декодировать путём нажатия клавиши "deshifr".
